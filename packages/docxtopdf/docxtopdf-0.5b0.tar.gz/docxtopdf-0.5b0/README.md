# docxtopdf

## Para usar...

### Windows:

Instalar o pacote Office da Microsoft.

### Linux:

Instalar o Libreoffice.

## Argumentos
| Argumento | Descrição |
| - | - |
| -h, --help | Mostra a mensagem de ajuda. | 
| --docx | Arquivo docx. |
| --output | Saída do pdf. | 
| --debug | Debug ||
